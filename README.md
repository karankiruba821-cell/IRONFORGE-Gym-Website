# 💪 IRONFORGE Gym Website

## 📌 Project Overview

**IRONFORGE Gym Website** is a modern and responsive fitness website built using **HTML, CSS, and JavaScript**.
The website promotes a gym brand, showcases training programs, trainers, schedules, and membership plans.

It is designed with a **dark premium UI**, smooth animations, and interactive elements to create a professional gym landing page.

---

## 🚀 Features

* Modern gym landing page design
* Responsive navigation menu with hamburger button
* Smooth scrolling navigation
* Animated hero section
* Training programs showcase
* Weekly class schedule with tab switching
* Trainers showcase section
* Membership pricing plans
* Member testimonials
* Contact form
* Scroll reveal animations
* Interactive navigation highlighting

---

## 🛠️ Technologies Used

* **HTML5** – Website structure
* **CSS3** – Styling, layout, animations, and responsive design
* **JavaScript (Vanilla JS)** – Interactivity and dynamic behavior

## 📑 Website Sections

The website contains multiple sections:

* **Hero Section** – Gym introduction and call-to-action
* **About Section** – Gym history and facilities
* **Programs Section** – Available training programs
* **Schedule Section** – Weekly class timetable
* **Trainers Section** – Professional trainer profiles
* **Pricing Section** – Membership plans
* **Testimonials Section** – Member success stories
* **CTA Section** – Free trial promotion
* **Contact Section** – Location, phone, email, and contact form
* **Footer** – Additional links and social media

---

## ✨ JavaScript Functionality

The website includes several interactive JavaScript features:

* Dynamic navbar style on scroll
* Active navigation link highlighting
* Mobile hamburger menu toggle
* Tab switching for weekly schedule
* Scroll reveal animations using Intersection Observer
